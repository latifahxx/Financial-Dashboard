# Financial Dashboard

## Project Overview
Monitors financial transactions by department, category, and region to improve budget allocation and provide financial visibility.

## Tools Used
- Excel (Pivot Tables, Charts)
- Power BI (Dashboards, KPIs, Data Visualization)

## Features
- Tracks revenue and expenses by category and department
- Financial summaries and trend analysis
- Budget comparison and reporting

## Impact
- Improved budget monitoring and resource allocation
- Provided actionable insights for cost management
- Supported financial decision-making across departments

## How to Use
1. Open dataset in `/Data`
2. Explore dashboards in Power BI (`/Reports`)
3. Filter by category, department, or region
